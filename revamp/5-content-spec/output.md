## 共用寫作指南

### 品牌語氣

| 維度 | 規格 | 範例 |
|------|------|------|
| 正式度 | 專業正式（智庫/官方統計機構風格） | "美國對台灣貿易逆差達 USD 1,269 億" |
| 專業度 | 使用專業術語，首次出現時附帶解釋 | "HHI 指數（Herfindahl-Hirschman Index，市場集中度指標）" |
| 情感 | 理性、中立、數據驅動 | 避免 "爆炸性增長！" "驚人！" |
| 人稱 | 第三人稱為主，必要時使用「本站」 | "本站追蹤六大經濟體貿易數據" |

### 用語規範

| 使用 | 避免 |
|------|------|
| 數據顯示 / 數據指出 | 數據證明 / 數據確認 |
| 較上期增長 X% | 暴漲 / 飆升 |
| 較上期減少 X% | 暴跌 / 崩盤 |
| 值得關注 / 需持續觀察 | 令人擔憂 / 危機 |
| 推測 / 可能 | 肯定會 / 必然 |
| 建議人工審核 | 絕對正確 |
| USD 1,269 億 | 1269億美元 / $126.9B |
| 2025 全年 | 25年 / 去年 |
| 台灣、美國、中國 | 台、美、中 |
| HHI（首次出現附全稱） | 直接使用縮寫 |
| UN Comtrade / US Census | Comtrade / Census |

### 數值格式規範

| 類型 | 格式 | 範例 |
|------|------|------|
| 金額（億美元級） | USD X,XXX 億 | USD 1,269 億 |
| 金額（兆美元級） | USD X.XX 兆 | USD 1.03 兆 |
| 百分比 | X.XX% 或 X% | 72%、54.08% |
| 年度 | YYYY 全年 / YYYY-MM | 2025 全年、2026-02 |
| 週別 | YYYY-WXX / 第 XX 週 | 2026-W08、第 08 週 |
| 日期範圍 | MM-DD ~ MM-DD | 02-17 ~ 02-23 |
| 國家代碼 | 全稱（括號內附代碼） | 台灣（158）、美國（842） |

### 格式規範

| 項目 | 規格 |
|------|------|
| 段落長度 | 最多 4-5 行，超過則分段 |
| 句子長度 | 最多 40 字（中文），超過則斷句 |
| 列表使用 | 超過 3 項時使用列表 |
| 強調方式 | 粗體（`**文字**`）用於關鍵數據；`<span class="data-highlight">` 用於報告內重要數據 |
| 引用區塊 | `>` 用於來源說明、注意事項 |
| 表格 | 數據比較、索引清單 |

### 連結規範

| 類型 | 格式 | 範例 |
|------|------|------|
| 內部報告連結 | `[報告名稱]({% link Narrator/mode/file.md %})` | `[查看]({% link Narrator/trade_briefing/2026-W08-trade-briefing.md %})` |
| 外部數據源 | 完整 URL + 明確來源名稱 | `[UN Comtrade](https://comtradeplus.un.org/)` |
| 錨點連結 | `[文字](#section-id)` | `[詳見市場集中度](#hhi-table)` |

## 1. 頁面目標

### 主要目標

3 秒內讓新訪客理解網站價值、5 秒內找到適合的報告入口

### 次要目標

1. 建立「數據可追溯、專業可信」的品牌印象
2. 展示最新重點內容，吸引深入閱讀
3. 提供訂閱入口，建立回訪機制

### 成功指標

| 指標 | 目標值 | 測量方式 |
|------|--------|----------|
| 跳出率 | < 50% | Google Analytics |
| 平均停留時間 | > 2 分鐘 | Google Analytics |
| 每次瀏覽頁數 | > 1.5 頁 | Google Analytics |
| RSS 訂閱數 | Phase 1: > 10 | RSS feed 統計 |

##### emoji 圖例規格（新增）

必須在「本週重點」表格上方新增以下圖例說明：

```html
{: .note }
> **圖例說明**：🔺 新增風險 ｜ ⚠️ 持續警示 ｜ 🔻 風險降低 ｜ 🔄 進行中/觀察
```

| emoji | 意義 | 使用情境 |
|:-----:|------|----------|
| 🔺 | 新增風險 | 本週新出現或顯著惡化的風險因素 |
| ⚠️ | 持續警示 | 已存在且持續需關注的風險 |
| 🔻 | 風險降低 | 風險緩解或正面發展 |
| 🔄 | 進行中/觀察 | 事件發展中，尚無定論 |

## 2. AEO 標記規範

### 必須實作的 CSS Class

| CSS Class | 用途 | 使用位置 |
|-----------|------|----------|
| `.key-answer` | 回答核心問題的段落 | 每個 H2 標題上，搭配 `data-question` 屬性 |
| `.key-takeaway` | 重點摘要區塊 | 摘要區塊，用 `<div>` 包裹 |
| `.article-summary` | 文章摘要 | 摘要區塊的 H2 或 `<div>` |
| `.speakable-content` | 可被語音助手朗讀的內容 | 與 `.article-summary` 同時使用 |
| `.data-highlight` | 重要數據標示 | 用 `<span>` 包裹重要數字 |
| `.comparison-table` | 比較表格 | 含數據比較的表格 |
| `.actionable-steps` | 行動建議清單 | 用 `<div>` 包裹觀察清單或行動建議 |

### `.key-answer` 使用範例

```markdown
## 一、出口重點
{: .key-answer data-question="2026年第8週各國出口情況如何？"}
```

每個報告的主要 H2 標題都必須搭配 `.key-answer` 和對應的 `data-question`。

### `.key-takeaway` 使用範例

```html
<div class="key-takeaway" markdown="1">
本週數據更新涵蓋六大報告國。**本週重點**：（1）<span class="data-highlight">美國對台灣貿易逆差達 USD 1,269 億</span>；（2）...
</div>
```

### `.data-highlight` 使用範例

```html
台灣（158）出口數據維持高可信度。前三大出口市場為中國（<span class="data-highlight">USD 1,926 億</span>）、美國（<span class="data-highlight">USD 1,878 億</span>）...
```

### `.comparison-table` 使用範例

```markdown
| 國家 | HHI 指數 | 集中度 |
|------|---------|--------|
| 中國 | 447.18 | 低 |
| 台灣 | 1183.15 | 低（接近中度） |
{: .comparison-table}
```

### `.actionable-steps` 使用範例

```html
<div class="actionable-steps" markdown="1">
1. **美台貿易逆差持續擴大關注** — 美國對台灣貿易逆差...
2. **中日關係緊張持續** — 2026-01-06 中國禁止...
</div>
```
