## 數據流程

```mermaid
flowchart LR
    A[數據源] --> B[自動擷取]
    B --> C[NLP 萃取]
    C --> D[向量儲存]
    D --> E[報告產出]
    E --> F[人工審核]
    F --> G[發布]

    style A fill:#e1f5fe
    style G fill:#c8e6c9
```

### 處理階段說明

| 階段 | 說明 | 自動化程度 |
|------|------|-----------|
| 數據擷取 | 定時從各 API 擷取最新數據 | 100% |
| NLP 萃取 | Claude 分析政策文件，萃取關鍵訊息 | 100% |
| 向量儲存 | 萃取結果存入 Qdrant 向量資料庫 | 100% |
| 報告產出 | 依據各 Mode 框架整合數據 | 100% |
| 人工審核 | 趨勢解讀與異常數據確認 | 0% |

{: .warning }
> 自動化程度標記為 80%：數據彙整完全自動化，但趨勢解讀與異常數據建議人工審核。
