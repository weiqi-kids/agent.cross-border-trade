# 荷兰扩大对先进半导体生产设备的出口管制措施

## Summary

2024年9月6日，荷兰政府网站发布消息，自9月7日起，荷兰将扩大针对先进半导体生产设备的国家出口管制措施。从该日起，更多种类的半导体生产设备的出口将需要申请许可证。新的许可证要求是对自2023年9月1日生效的国家出口管制措施的扩展。
