# 日本修改出口管制清單新增半導體增材製造物項並修訂全面管制政策 (Japan Amends Export Control List Adding Semiconductor and Additive Manufacturing Items and Revises Catch-All Control Policy)

**Source:** 中國出口管制信息網
**URL:** http://exportcontrol.mofcom.gov.cn/article/gjdt/202502/1103.html
**Date:** 2025-02-12
**Category:** regulation_update
**Fetched:** 2026-01-28
**Confidence:** High

## Summary

日本政府宣布修改出口管制清單，新增半導體、增材製造等相關物項。同日發布修訂草案，對日本出口管制制度中的"全面管制"(catch-all control) 相關政策進行修訂。日本政府就上述修訂征求公眾意見，截止日期為 2025-03-01。

## Key Details

- **Issuing Authority:** 日本政府 (Government of Japan)
- **Effective Date:** 待公眾意見征詢結束後確定（公眾意見截止日期：2025-03-01）
- **Scope:** 出口管制清單及全面管制政策修訂

## Key Provisions

1. **出口管制清單修訂**：
   - 新增半導體 (semiconductors) 相關物項
   - 新增增材製造 (additive manufacturing) 相關物項

2. **全面管制政策修訂**：
   - 對日本出口管制制度中的"全面管制"相關政策進行修訂
   - 全面管制是指對未列入管制清單但可能用於大規模殺傷性武器或常規武器開發的物項實施的出口許可要求

3. **公眾意見征詢**：
   - 征詢截止日期：2025-03-01
   - 顯示日本政府在出口管制政策制定中的透明度和民主程序

## Supply Chain Impact Assessment

**Direct Impact** (speculative):
- 日本半導體和增材製造相關產品出口將面臨更嚴格管制
- 可能需要申請出口許可或進行最終用戶審查
- 全面管制政策修訂可能擴大管制範圍，影響更多未列入清單的物項

**Indirect Impact** (speculative):
- 反映日本配合國際社會加強對先進技術出口的管制
- 可能與美國、歐盟等盟友的出口管制政策協調一致
- 全球半導體和增材製造供應鏈面臨更多合規挑戰

**Strategic Implications** (speculative):
- 日本出口管制政策持續收緊，特別是針對新興技術領域
- 全面管制政策修訂可能針對特定國家或最終用戶加強審查
- 中國企業在從日本進口半導體和增材製造相關技術時需關注政策變化
- 日本可能進一步限制對中國的先進技術出口
