# 瑞士修訂《貨物管制條例》新增新興技術領域兩用物項管制 (Switzerland Amends Goods Control Ordinance to Add Emerging Technology Dual-Use Items)

**Source:** 中國出口管制信息網
**URL:** http://exportcontrol.mofcom.gov.cn/article/gjdt/202504/1127.html
**Date:** 2025-04-08
**Category:** regulation_update
**Fetched:** 2026-01-28
**Confidence:** High

## Summary

瑞士聯邦委員會通過《貨物管制條例》修正案，新增針對半導體、量子計算、增材制造等新興技術領域的兩用物項出口管制規定。此舉旨在確保瑞士出口管制措施與國際接軌，並協調與重要貿易夥伴的出口管制制度。修訂將於 2025-05-01 起生效。

## Key Details

- **Issuing Authority:** 瑞士聯邦委員會 (Swiss Federal Council)
- **Effective Date:** 2025-05-01
- **Scope:** 新興技術領域兩用物項出口管制

## Key Provisions

1. **修訂通過**：
   - 瑞士聯邦委員會於 2025-04-02 通過《貨物管制條例》修正案

2. **新增管制物項**：
   - 半導體 (semiconductors) 相關物項
   - 量子計算 (quantum computing) 相關物項
   - 增材製造 (additive manufacturing) 相關物項

3. **政策目標**：
   - 確保瑞士出口管制措施繼續與國際接軌
   - 協調與重要貿易夥伴的出口管制制度

4. **生效日期**：
   - 2025-05-01 起正式實施

## Supply Chain Impact Assessment

**Direct Impact** (speculative):
- 瑞士出口商在出口半導體、量子計算、增材製造相關物項時需遵守新的管制要求
- 可能需要申請出口許可或進行最終用戶審查
- 瑞士在相關高科技領域的出口流程將更加嚴格

**Indirect Impact** (speculative):
- 反映國際社會對新興技術擴散風險的普遍關注
- 瑞士作為傳統中立國，其管制政策調整可能影響其他歐洲國家
- 全球新興技術供應鏈面臨更多合規要求
- 可能促使企業加強出口管制合規體系建設

**Strategic Implications** (speculative):
- 新興技術領域成為全球出口管制的新焦點
- 國際出口管制協調機制（如瓦森納安排）可能已就新興技術達成共識
- 瑞士政策調整顯示西方國家在出口管制領域的政策趨同
- 中國企業在與瑞士開展相關技術合作時需更加關注合規風險

## Notes

本文為中國出口管制信息網轉載的國際動態報導。瑞士作為傳統中立國和重要高科技產品出口國，其出口管制政策調整具有重要指標意義。文章未提供具體管制物項的詳細清單，但明確提及半導體、量子計算、增材製造三大新興技術領域，反映了國際社會對這些前沿技術擴散風險的共同關切。
