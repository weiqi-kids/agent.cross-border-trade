**綜合評估**：

中國自 2023 年石墨管制開始，已形成涵蓋稀土、鋰電池材料、關鍵礦物（鎵鍺銻鎢碲鉍鉬銦）的完整出口管制體系。管制策略有三個特點：

1. **涵蓋生產技術**：不僅管制原材料，同時限制生產設備與製程技術出口，阻礙進口國建立替代供應鏈
2. **境外延伸效力**：「含中國成分之外國稀土物項」管制開創境外再出口審批先例
3. **國別差異化**：對美國實施「原則拒絕」政策，對盟國維持許可審批制度

**詳細政策內容**：

**稀土技術管制（公告 61、62 號）**：
- 管制「含中國成分的外國稀土相關物項」，具域外效力
- 稀土技術早於 2001 年已列入限制出口技術目錄
- 人道主義豁免：緊急醫療、公共衛生、救災用途免許可
- 設有合理過渡期與許可便利措施

*來源：cn_export_control/mofcom-rare-earth-export-control-strengthening-oct2025.md*

**中重稀土管制（公告 57 號）**：
- 涵蓋五類重稀土：鈥(Ho)、鉺(Er)、銩(Tm)、銪(Eu)、鐿(Yb)
- 管制形態包括金屬、合金、靶材、晶體材料、發光材料、光纖材料、永磁體及化合物
- 影響產業：半導體靶材、EDFA 光纖放大器、OLED/LED 顯示器、醫療/軍用雷射

*來源：cn_export_control/heavy-rare-earth-export-controls.md*

**鋰電池材料管制（公告 58 號）**：
- 三類管制物項：
  - 高能量密度鋰電池（≥300 Wh/kg）及製造設備
  - 磷酸鐵鋰正極材料（密度 ≥2.5 g/cm³）及生產系統
  - 人造石墨負極材料及石墨化爐、塗覆改質設備
- 生效日：2025-11-08

*來源：cn_export_control/lithium-battery-graphite-anode-export-controls.md*

**鎢碲鉍鉬銦管制（公告 10 號）**：
- 鎢：碳化鎢、高密度鎢合金（≥17.0 g/cm³）
- 碲：碲化鎘、碲化鋅鎘、碲化汞鎘（紅外線偵測器材料）
- 鉍：鍺酸鉍（Bi₄Ge₃O₁₂）、三苯基鉍
- 鉬：高純度鉬粉（≥97%、≤50 μm）用於飛彈元件
- 銦：磷化銦、三甲基銦、三乙基銦（III-V 族半導體）
- 生效日：2025-02-04（公布即生效）

*來源：cn_export_control/critical-minerals-export-control-2025.md*

### 計算方法

- **HHI**：Herfindahl-Hirschman Index，市場份額平方和（0-10000），<1500 低集中度、1500-2500 中等、>2500 高集中度
- **YoY**：Year-over-Year，同比變化率
- **市場份額**：個別夥伴國/商品占總量比例
- **語意搜尋**：使用 OpenAI text-embedding-3-small 模型產生向量，Qdrant 餘弦相似度匹配

### 已知限制

1. **Comtrade 數據延遲**：2024 年全年數據為最新可用，2025 年數據尚未完整發布
2. **Census 初報值修正**：月度數據可能在後續發布中修正 1-3%
3. **政策影響不確定性**：出口管制實際執行力度與許可核發情況難以即時追蹤
4. **HS 商品分類限制**：Preview API 未提供 HS6 級別細部分類，商品集中度分析為推估
5. **替代供應來源產能**：非中國產能數據來自產業報告，更新頻率較低
