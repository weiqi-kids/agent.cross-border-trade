## 數據相關

### 數據來源是否可靠？
{: .key-answer data-question="數據來源是否可靠？"}

所有數據均來自公開、權威的國際組織與政府機構：

| 數據源 | 提供者 | 可信度 |
|--------|--------|--------|
| UN Comtrade | 聯合國 | ✅ 全球最權威貿易統計 |
| US Census Bureau | 美國商務部 | ✅ 美國官方數據 |
| World Bank | 世界銀行 | ✅ 全球宏觀經濟權威 |
| 中國商務部 | 中國政府 | ✅ 中國官方政策來源 |

{: .highlight }
> 每項數據都標註原始來源，您可自行驗證。

---

### 數據更新頻率為何？
{: .key-answer data-question="數據更新頻率為何？"}

**報告產出頻率**：
- 週報：每週一
- 月報：每月 5 日前
- 季報：每季初第一週

**原始數據更新**：
- UN Comtrade：年度/季度，滯後約 2 個月
- US Census：月度，滯後約 6 週
- 中國商務部：政策公告即時更新

詳見 [數據更新時間軸](/data-timeline/)。

---

### 為何只涵蓋六個國家？
{: .key-answer data-question="為何只涵蓋六個國家？"}

我們聚焦**台灣、美國、中國、日本、韓國、德國**這六大經濟體，原因：

1. **全球貿易核心**：六國合計占全球貿易總額約 40%
2. **供應鏈關鍵節點**：半導體、汽車、機械產業的核心玩家
3. **政策影響力大**：美中貿易政策、中國出口管制影響全球
4. **深度優於廣度**：聚焦讓我們能提供更深入的分析

{: .note }
> 未來可能擴展至 G20 或東協國家，但會確保不犧牲分析深度。

### HHI 指數是如何計算的？
{: .key-answer data-question="HHI 指數是如何計算的？"}

**Herfindahl-Hirschman Index (HHI)** 計算公式：

```
HHI = Σ(市場份額²) × 10,000
```

**範例**：某國前三大出口市場各占 30%、20%、10%，其他占 40%

```
HHI = (0.30² + 0.20² + 0.10² + 0.40²) × 10,000
    = (0.09 + 0.04 + 0.01 + 0.16) × 10,000
    = 3,000
```

**判讀標準**：

| HHI 範圍 | 集中度 | 風險評估 |
|----------|--------|---------|
| < 1,500 | 低 | 市場分散，風險較低 |
| 1,500-2,500 | 中 | 需關注主要市場變動 |
| > 2,500 | 高 | 高度依賴，需多元化 |

### 為何是開源架構？
{: .key-answer data-question="為何是開源架構？"}

開源的三大理由：

1. **透明可審核**：方法論、計算邏輯完全公開，專業人士可審核
2. **可複製性**：其他研究者可基於本架構建立類似系統
3. **社群貢獻**：歡迎提交 Issue 或 Pull Request 改善系統
